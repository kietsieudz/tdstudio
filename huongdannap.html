Bạn muốn nạp cho server nào?

Bước 1: Truy cập trang nạp
Truy cập vào ThanhDi.Com → chọn Đăng ký, Tải game → chọn Server bạn muốn nạp.
Bước 2: Vào trang chủ server game
Bấm vào nút Đăng nhập → đăng nhập tài khoản → bấm Diễn đàn.
Bước 3: Vào diễn đàn & chọn nạp
Trong diễn đàn sẽ hiển thị nút Nạp thẻ. Hình thức Bank hay Card đều giống nhau.
Bước 4: Chọn loại nạp & thanh toán
Chọn loại nạp phù hợp → tiến hành thanh toán → chờ xử lý.
Coin sẽ vào web trước (3–15 phút). Khi coin đã có, thoát game và vào lại để cập nhật trong game.
Bước 5: Quy đổi trong game
Sau khi coin vào web → Hãy thoát game và Đăng nhập lại → Gặp NPC trong nhà → Nạp tiền & Quy đổi → đổi sang Hồng Ngọc hoặc Thỏi Vàng.
Bước 6: Nhận quà nạp
Nhớ nhận quà tặng và quà mốc nạp tại NPC.
Bước 7: Gặp lỗi & báo lỗi
Trường hợp quên nội dung chuyển khoản hoặc lỗi nạp, vui lòng bấm ĐÂY để liên hệ hỗ trợ.
