Hướng dẫn trang bị người chơi mới
Dành cho người lần đầu tham gia game
1. Game có nhiều trang bị khác nhau
  • Top 7: Set kích hoạt thường
  • Top 6: Set kích hoạt thần linh
  • Top 5: Set kích hoạt Thiên sứ
  • Top 4: Set kích hoạt Thần tiên
  • Top 3: Set kích hoạt Hạ Tôn
  • Top 2: Set kích hoạt Chí Tôn
  • Top 1: Set kích hoạt Vampa
2. Sét kích hoạt thường
  • Nhập từ Giftcode tân thủ sẽ có
  • Hoặc đánh quái 3 map cạnh làng sẽ rơi
3. Set kích hoạt thần linh
  • Đến gặp Quy lão vào khu Vực bang gặp Giu-ma và chọn đến thí luyện cá nhân (đạt được thẻ ở đó sẽ đổi được set thần linh kích hoạt)
  • Vào lúc 21:00 hàng ngày đến gặp Osin ở Đại hội võ thuật (làng kakarot đi qua bên trái) để đến Đại chiến Mabư kiếm hạt đậu rồi ra Santa đổi set Thần linh kích hoạt
4. Set kích hoạt Thiên sứ
  • Cần săn boss hoặc up cold để kiếm trang bị thần linh
  • Mang trang bị thần linh lên Thần hủy diệt để đổi trang bị Hủy diệt
  • Mang trang bị thần linh và hủy diệt đổi trang bị Thiên sứ ở Thiên Sứ whis ở Hành tinh Kaio
  • Mang trang bị thần linh và hủy diệt và thiên sứ ra Bà Hạt Mít đảo kame để nâng lên trang bị Thiên sứ kích hoạt
5. Sét kích hoạt Thần tiên
  • Bạn cần phiếu giảm giá để đổi Rương trang bị thần tiên kích hoạt tại Santa
  • Săn boss Doraemon, săn boss Liễu Thần để nhặt phiếu giảm giá
  • Đi con đường rắn độc (ở Thần vũ trụ tại hành tinh Kaio) để nhặt phiếu giảm giá
  • Và một vài event khác sẽ có phiếu giảm giá
6. Sét kích hoạt Hạ tôn
  • Chế tạo set kích hoạt hạ tôn tại Bà Hạt Mít đảo kame
  • Đọc nguyên liệu tại Bà Hạt Mít
7. Sét kích hoạt Thiên tôn (Chí tôn)
  • Chế tạo set kích hoạt hạ tôn tại Bà Hạt Mít đảo kame
  • Đọc nguyên liệu tại Bà Hạt Mít
  • Mua rương Chí tôn tại Santa (Mở ra 1 set 5 món kích hoạt, được lựa chọn set, không random)
8. Sét kích hoạt Vampa
  • Khi bạn đạt được 1 set thiên tôn 12 sao hãy đến Vách núi đen gặp Paragus để đến Hành tinh Vampa
  • Tiêu diệt Broly và Super Broly tại Hành tinh Vampa sẽ rơi trang bị Vampa ngẫu nhiên (không thể giao dịch)
  • Hoặc đạt top 1, top 2 nạp thẻ hàng tháng sẽ được trang bị Vampa max (Mọi thứ đều đã max sẵn, đục lỗ, nâng cấp level,... Chỉ cần nhận rồi dùng không cần làm gì nữa) Xem chi tiết
